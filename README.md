🎮 Theme & Design

This portfolio follows a Game UI / Space / Retro futuristic theme with:

- Twinkling star background ✨
- Floating glow blobs 🌌
- Glassmorphism cards 🧊
- Animated UI interactions 🎯
- Game-style panels and hover effects

📄 Website Pages
Page	Description
Home (index.html)	Introduction, about me, skills, and overview
Projects (projects.html)	Details about my portfolio project and links
Contact (contact.html)	Contact form connected to backend & MongoDB
GameDev (gamedev.html)	My interest in open-world and immersive game development

🛠️ Tech Stack Used
Frontend
- HTML5
- CSS3 (Animations, Glassmorphism, Game UI styling)
- JavaScript (Fetch API)

Backend
- Node.js
- Express.js
  
Database
- MongoDB Atlas
Deployment

- Frontend → Netlify
- Backend → Render
Version Control
- Git & GitHub

🔄 Modern Web Development Workflow Followed
1. Designed UI using HTML & CSS
2. Added interactivity using JavaScript
3. Created backend server using Node & Express
4. Connected contact form to MongoDB Atlas
5. Pushed code to GitHub repository
6. Deployed frontend on Netlify
7. Deployed backend on Render
8. Tested live integration between frontend and backend

📬 Contact Form Functionality
The contact page allows users to send messages which are:

- Captured using JavaScript Fetch API
- Sent to Express backend hosted on Render
- Stored in MongoDB Atlas database
